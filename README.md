# 專利相關法規速查 · IP Law Lookup

一個輕量的單頁工具，讓使用者快速查詢台灣專利相關法規的條文，並同時顯示**中文、英文、日文**三語對照。

🔗 **線上版本：** https://ip-law-lookup.pages.dev/patent-related-law-lookup.html

---

## 功能

- 輸入條號、項、款，即時顯示對應條文
- 三語對照：中文原文 / 英文官方譯本 / 日文機器翻譯
- 支援複製條文與連結至官方法規資料庫
- 純靜態單檔，無需伺服器，可離線使用

---

## 涵蓋法規

| 法規 | 條數 |
|------|------|
| 專利法 Patent Act | 159 條 |
| 專利法施行細則 Enforcement Rules | 93 條 |
| TRIPS（與貿易有關之智慧財產權協定） | 73 條 |
| 智慧財產案件審理法 IP Case Adjudication Act | 77 條 |

---

## 使用方式

1. 從下拉選單選擇法規
2. 輸入條號（必填）、項（選填）、款（選填）
3. 點「查詢」

---

## 技術說明

- 純 HTML / CSS / JavaScript，零框架、零依賴
- 法條資料內嵌於單一 HTML 檔案
- 日文為機器翻譯（非官方），僅供參考
- 部署於 Cloudflare Pages

---

## 資料來源

- 中文法條：[全國法規資料庫](https://law.moj.gov.tw)
- 英文法條：[全國法規資料庫英文版](https://law.moj.gov.tw/ENG)
- TRIPS：[WTO 官方文本](https://www.wto.org/english/docs_e/legal_e/27-trips.pdf)
